🎯 Interpretação Geométrica

O determinante det = 5.0 > 0 indica que a transformação preserva a orientação. A área do paralelogramo formada pelos vetores é |det| = 5.0 unidades². A forma diferencial ω = 5dx ∧ dy representa o elemento de área escalado por este fator.

✨ Conexão Fundamental

Determinante ↔ Área ↔ Forma Diferencial
det(A) = área orientada = coeficiente de dx ∧ dy
A transformação linear x' = Ax multiplica áreas por |det(A)|

🧮 Cálculos Detalhados

Produto Vetorial (2D):
u₁ × u₂ = (2)(3) - (1)(1) = 6 - 1 = 5
Produto Exterior:
u₁ ∧ u₂ = (2dx + 1dy) ∧ (1dx + 3dy) = 2·3(dx ∧ dy) - 1·1(dy ∧ dx) = 5(dx ∧ dy)
Jacobiano da Transformação:
J = ∂(x',y')/∂(x,y) = det([∂x'/∂x, ∂x'/∂y; ∂y'/∂x, ∂y'/∂y]) = det = 5

K-Cadeias e Operador Bordo

C = σ₁ + σ₂ + σ₃ + σ₄ (2-cadeia como soma de superfícies orientadas)

Operador Bordo: O bordo de uma k-cadeia é a soma dos bordos individuais:

∂C = ∂σ₁ + ∂σ₂ + ∂σ₃ + ∂σ₄

Cancelamento: Quando duas superfícies adjacentes compartilham uma aresta, seus bordos contribuem com orientações opostas nessa aresta:

∂σᵢ = +e₁₂ + e₂₃ - e₃₄ - e₄₁ (exemplo de orientação) ∂σⱼ = -e₁₂ + e₂₅ + e₅₆ - e₆₁ (aresta compartilhada com sinal oposto)

Resultado: Na soma ∂C, arestas internas se cancelam: (+e₁₂) + (-e₁₂) = 0

∂²C = 0 (O bordo do bordo é sempre zero)

Interpretação Geométrica: O cancelamento de bordos internos é a razão matemática pela qual superfícies coladas formam uma única superfície com bordo contínuo.
